# TONBOT

TONBOT is a Python script for sending customized messages using the TON API.

## Get Your API KEY

To use this project, you'll need to obtain an API key from [TonConsole](https://tonconsole.com/). Follow these steps:

1. Go to [TonConsole](https://tonconsole.com/).
2. Sign up for an account or log in if you already have one.
3. Navigate to the API Key section.
4. Generate a new API key.
5. Copy the API key and keep it secure.
